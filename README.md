# Dicoding Submission Fundamental Front End Web Development

**CARA INSTALASI**

 1. Jalankan perintah

    npm install

 2. Menjalankan aplikasi menggunakan webpack production

    npm run build-web

 3. Atau menggunakan webpack development
 
    npm run start-web


**KRITERIA SUBMISSION**

Mendapat 4 Bintang dari 5.
